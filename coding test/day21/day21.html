Q. 문자열 my_string이 매개변수로 주어집니다. my_string안의 모든 자연수들의 합을 return하도록 solution 함수를 완성해주세요.

JAVASCRIPT 
function solution(my_string) {
  var answer = 0;
  var a = [...my_string.replace(/[^0-9]/g, "")].map((a)=> Number(a)).map((a)=>answer +=a).reverse()
  return a[0]
}

////////////// 다른 풀이 ///////////////

function solution(my_string) {
    return my_string.replaceAll(/[^\d]/g, '').split('').map(v=>+v).reduce((a,v)=>a+v,0);
}

Q. 문자열 my_string이 매개변수로 주어집니다. my_string은 소문자, 대문자, 자연수로만 구성되어있습니다.
my_string안의 자연수들의 합을 return하도록 solution 함수를 완성해주세요.

JAVASCRIPT 
function solution(my_string) {
  return my_string.split(/\D+/).reduce((a,c)=> a*1 +c*1);
}

문자열 my_string이 매개변수로 주어질 때, 대문자는 소문자로 소문자는 대문자로 변환한 문자열을 return하도록 solution 함수를 완성해주세요.

JAVASCRIPT 
function solution(my_string) {
  return my_string.split('').map(a=> a.match(/[A-Z]/g) ? 
  a.toLowerCase(): a.toUpperCase()).join("")
}

i팩토리얼 (i!)은 1부터 i까지 정수의 곱을 의미합니다. 예를들어 5! = 5 * 4 * 3 * 2 * 1 = 120 입니다.
정수 n이 주어질 때 다음 조건을 만족하는 가장 큰 정수 i를 return 하도록 solution 함수를 완성해주세요. i! ≤ n

JAVASCRIPT 
function solution(n) {
  let i = 1;
  let f = 1;
  while (f*i < n) f*=++i;
  return console.log(i);
}
