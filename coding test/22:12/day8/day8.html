Q. 첫째 줄에는 테스트 케이스의 개수 C가 주어진다. 둘째 줄부터 각 테스트 케이스마다 학생의 수 N(1 ≤ N ≤ 1000, N은 정수)이 첫 수로 주어지고,
이어서 N명의 점수가 주어진다. 점수는 0보다 크거나 같고, 100보다 작거나 같은 정수이다.

JAVASCRIPT 


//// 예제
5
5 50 50 70 80 100
7 100 95 90 80 70 60 50
3 70 90 80
3 70 90 81
9 100 99 98 97 96 95 94 93 91 //


// 출력
40.000%
57.143%
33.333%
66.667%
55.556% //

const input = require('fs').readFileSync(0).toString().trim().split('\n');
input.splice(0,1)
let newSet = input.map(a=>a.split(' '))
let people = newSet.map(a=>a.splice(0,1))
let average = newSet.map((a,i)=>eval(a.join('+'))/people[i])
let grade = [...newSet]
let array = []

for (let i=0; i < newSet.length; i++){
  let count = 0;
  for (let j=0; j < grade[i].length; j++){
    grade[i][j] > average[i] ? count+=1 : null;
  }
  array.push(count)
}

array.map((a,i)=>console.log((a/people[i]*100).toFixed(3)+'%'));

[백준 반복문-2439번]
Q. 첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 문제
하지만, 오른쪽을 기준으로 정렬한 별(예제 참고)을 출력하시오.

JAVASCRIPT 
const input = require('fs').readFileSync(0).toString();
let star = '';

for (let i = 0; i < input; i++){
  for (let j = input-1; j>=0; j--){
    star += j<=i ? '*' : ' ';
  }
  star += '\n'
}
console.log(star)
