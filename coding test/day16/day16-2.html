Q. 문자열 변수를 array로 어떻게 변환해야할까?

JAVASCRIPT 

var a = "daeun"

var b = [...a]
var c = a.split("")

Q. 반대로 array를 문자열 변수로 변환하려면?

JAVASCRIPT 

var a = ['d', 'a', 'e', 'u', 'n']

var b = a.join("");
var c = a.toString().replace(/,/g, "");

Q. array안의 문자열을 숫자열로 변환하려면?

JAVASCRIPT 

var a = ['d', 'a', 'e', 'u', 'n']

var b = a.map((a)=> Number(a));

Q. 숫자를 배열로 변환시키려면?

JAVASCRIPT 

var a = 1234

var b = String(a).split('') //문자열로 변환하여 array를 생성한다.
