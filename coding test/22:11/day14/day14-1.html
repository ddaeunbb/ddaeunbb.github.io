#reverse 문자열, array 반대정렬하기

📍reverse API

Q. 문자열 my_string이 매개변수로 주어집니다. my_string을 거꾸로 뒤집은 문자열을 return하도록 solution 함수를 완성해주세요.

JAVASCRIPT 
function solution(my_string){
  var answer = 0;
  answer = my_string.split('').reverse().join('');
  return answer
}

Q. 그럼 array 관련해서 reverse는 어떻게 해야할까?

JAVASCRIPT 
function solution(n){
  var answer = [a, b, c, d, e]
  answer = answer.reverse();
  return answer;
}
