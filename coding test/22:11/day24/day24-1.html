#array, object key, value #Object.entries

Q. Array=[a,b,c] 를 key값으로 주고 value값으로 [1,2,3]을 주려면 어떻게 해야할까?

1) new Map과 반복문을 활용하여 만들기


JAVASCRIPT 

  const array = ['a','b','c']
  var newArray = new Map();
  for (i=0; i < array.length; i++;){
    newArray.set(array[i], i+1)
  }

  // 결과 값 {'a' => 1, 'b' => 2, 'c' => 3}

2) object와 반복문을 활용하여 만들기


JAVASCRIPT 

  const alphabet = "abcdab"
  var object = {}
  alphabet.split('').map(a=>object[a] !== undefined ? object[a]+=1 : object[a]=1)
  // 결과값 : {a: 2, b: 2, c: 1, d: 1}

Q. object에서 key와 value 값을 array로 가져오려면? + object key와 value를 배열로 한꺼번에 가져오려면?

JAVASCRIPT 

  var object = {a: 1, b: 2, c: 3, d: 4}

  console.log(Object.keys(object))  // 결과값 : ['a','b','c','d']
  console.log(Object.values(object))  // 결과값 : [1, 2, 3, 4]
  console.log(Object.entries(object)) // 결과값 : 0 : ['a',1], 1:['b',2], 2:['c',3]....
